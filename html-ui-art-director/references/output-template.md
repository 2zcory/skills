# Output Template

Use this shape when presenting a design artifact or design direction.

## Preferred output

- `Screen job`
- `Visual language`
- `Layout archetype`
- `Why this archetype`
- `Hero vs support`
- `Asset strategy`
- `Artifact files`
- `Critique score`
- `Rationale`

## Example shape

```text
Screen job
- Help product teams understand one urgent operating issue and act on it.

Visual language
- Calm premium / product-first

Layout archetype
- Split decision

Why this archetype
- The user needs belief and preview at the same time; a pure editorial sequence would hide the live product surface too long.

Hero vs support
- Hero: live workspace preview and primary message
- Support: metrics strip and one explanatory panel

Asset strategy
- Product-first preview
- Shape-led placeholder brand mark
- No decorative photography

Artifact files
- index.html
- styles.css

Critique score
- 31/35

Rationale
- Keeps one dominant path, uses the preview as the trust asset, and avoids card soup in the supporting regions.
```

## Notes

- Keep the rationale short.
- The score should reflect the rubric, not personal mood.
- If assumptions were made, note them before or inside the rationale.
